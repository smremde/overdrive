The first block of the filesystem starts with the so-called Supermeta header. After the SuperMeta Header comes a MetaGroupHeader, and the rest of the logical block is filled with MetaGroup items.

SuperMeta Header
Size (bytes) Description
4 Magic bytes (= 0x0DBF5001)
4 BlockSize

MetaGroup Header
Size (bytes) Description
8 BlockNumber of next MetaGroup

MetaGroup item
Size (bytes) Description
16 VBlockId
2 Total number of parts of this VBlock.
2 Part index number of this MetaGroupItem
8 BlockNumber of first datablock of this part
4 Number of blocks in this part
